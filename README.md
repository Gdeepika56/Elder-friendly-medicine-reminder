# Elder-friendly Medicine Reminder

An Android application designed to help users—especially elderly people—remember to take their medicines on time.

The app focuses on simplicity, clear visuals, and voice-based reminders to support users who may not be comfortable with complex smartphone interfaces.

## Features
- Add medicine name, dosage, and scheduled time
- Automatic reminders using WorkManager
- Notification alerts
- Voice reminders using Text-to-Speech
- Simple, elder-friendly UI design
- Mark medicines as taken
- Edit or delete medicines easily

## Tech Stack
- Kotlin
- Jetpack Compose
- Room Database
- WorkManager
- Reminder + Voice Notifications
- Text-to-Speech

## Status
Actively improving based on real user feedback.
